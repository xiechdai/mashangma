# 马上码 App 演示指南

## 应用特性展示

### 1. 主要功能演示

#### 📋 剪贴板自动读取
1. 复制任意文本到剪贴板
2. 打开"马上码"应用
3. 应用自动读取剪贴板内容
4. 智能识别编码类型并生成对应码

#### 🧠 智能编码识别
1. **URL内容** → 自动生成二维码
2. **纯数字(8-13位)** → 自动生成条形码
3. **中文内容** → 自动生成二维码
4. **长文本** → 自动生成二维码

#### 🔄 编码类型切换
1. 主界面底部横向滑动选择编码类型
2. 支持12种编码格式
3. 切换时立即重新生成码
4. 不兼容内容显示错误提示

#### 📜 历史记录功能
1. 点击左上角历史按钮查看编码历史
2. 点击右下角浮动按钮查看剪贴板记录
3. 支持搜索功能
4. 支持删除单条记录和批量清空

#### ✏️ 手动输入与编辑
1. 点击右上角输入按钮
2. 支持多行文本输入
3. 实时字符计数和内容验证
4. 快速粘贴和时间插入功能

#### 💾 保存与分享
1. 长按码图或点击保存按钮
2. 保存高清PNG格式到相册
3. 支持系统分享功能
4. 可分享图片或原始文本

#### ⚙️ 设置页面
1. 剪贴板监听开关
2. 敏感词过滤设置
3. 历史记录上限设置
4. 编码类型和分辨率设置
5. 主题模式选择

### 2. 支持的编码类型

#### 二维码类型
- **QR Code** - 最常用，支持中文、URL
- **Data Matrix** - 小型高密度码
- **PDF417** - 证件车票常用
- **Aztec Code** - 交通票据

#### 条形码类型
- **Code 128** - 通用商品码
- **EAN-13** - 13位国际商品码
- **EAN-8** - 8位短商品码
- **UPC-A** - 12位北美商品码
- **Code 39** - 工业码制
- **Code 93** - 高密度工业码
- **ITF-14** - 物流包装码
- **Codabar** - 图书馆血库专用

### 3. 隐私与安全

#### 本地数据处理
- 所有数据仅存储在用户设备
- 无需网络权限，完全离线工作

#### 敏感内容保护
- 支持敏感词过滤
- 自动识别并标记敏感内容
- 用户可选择是否保存敏感信息

#### 权限透明化
- 明确说明每项权限用途
- 按需请求，最小权限原则
- 提供权限拒绝时的友好提示
