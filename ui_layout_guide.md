# UI布局调整说明

## 📱 新的界面布局

### 整体结构（从上到下）

```
┌─────────────────────────────────┐
│ 1. 编码显示区域 (最大区域)       │
│    • 二维码/条形码预览          │
│    • 智能检测标签                │
│    • 内容预览框                  │
│    • 空状态/错误状态提示          │
├─────────────────────────────────┤
│ 2. 输入区域 (中等区域)          │
│    • 输入框                      │
│    • 字符计数                    │
│    • 清空/粘贴按钮               │
├─────────────────────────────────┤
│ 3. 码类型选择区域 (底部)         │
│    • 下拉选择菜单                │
│    • 智能检测提示                │
└─────────────────────────────────┘
```

## 🎨 布局特点

### 1. **编码显示区域**（最上方）
- **占用空间**: Expanded(flex: 2) - 最大区域
- **用途**: 二维码/条形码的主要展示
- **特性**:
  - 居中显示编码图像
  - 标题显示当前码类型
  - 智能检测状态标签
  - 底部内容预览框

### 2. **输入区域**（中间）
- **占用空间**: 固定高度 + 16px 边距
- **用途**: 用户输入内容的主要区域
- **特性**:
  - 多行文本输入框
  - 动态提示文本
  - 字符计数显示
  - 清空/粘贴快捷操作

### 3. **码类型选择区域**（最下方）
- **占用空间**: 固定高度 + 16px 边距
- **用途**: 选择不同的码类型
- **特性**:
  - 下拉选择菜单
  - 图标区分码类型
  - 智能检测标识
  - 优先级排序

## 🔧 设计优化

### 视觉层次
1. **主要焦点**: 编码显示（最大视觉区域）
2. **次要焦点**: 输入区域（用户交互核心）
3. **辅助功能**: 码类型选择（配置选项）

### 间距处理
- 各区域间使用 Divider 分隔
- 统一 16px 外边距
- 内容区域使用适当内边距

### 响应式设计
- 编码区域自适应高度
- 输入区域保持合适大小
- 选择区域固定在底部

## 📐 空状态优化

### 新的空状态设计
- 圆形图标容器
- 突出的提示文本
- 支持说明边框
- 操作指引提示

### 错误状态优化
- 红色圆形警告图标
- 清晰的错误标题
- 具体错误说明
- 解决建议提示

### 1. **视觉引导**
- 编码显示最大化，突出核心功能
- 层次分明，操作流程清晰

### 2. **操作便利**
- 输入框居中，符合操作习惯

### 3. **信息展示**

## 📱 适配说明

### 不同屏幕尺寸
- **大屏幕**: 编码区域充分利用空间
- **小屏幕**: 保持布局比例，可滚动查看
- **横屏**: 编码区域自适应宽度

### 交互优化
- 触摸区域合理大小
- 下拉菜单操作便捷
- 输入框多行支持
