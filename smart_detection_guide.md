# 智能码类型检测功能

## 🎯 自动检测规则

### 剪贴板内容自动识别
应用会智能检测剪贴板内容，并自动选择最适合的码类型：

#### 1. CODE_128 自动切换
**条件**: 纯字母数字组合（不含空格和特殊字符）
```
✅ "ABC12345" → 自动切换到 CODE_128
✅ "PRODUCT2024" → 自动切换到 CODE_128  
✅ "SKU789" → 自动切换到 CODE_128
```

#### 2. 特定长度数字条码
**条件**: 纯数字且符合标准长度
```
✅ "12345678" (8位) → 自动切换到 EAN_8
✅ "123456789012" (12位) → 自动切换到 UPC_A
✅ "1234567890123" (13位) → 自动切换到 EAN_13
✅ "12345678901234" (14位) → 自动切换到 ITF_14
```

#### 3. QR_CODE 默认选择
**条件**: 包含空格、特殊字符或复杂内容
```
✅ "Hello World!" → 保持 QR_CODE
✅ "https://example.com" → 保持 QR_CODE
✅ "用户名: admin, 密码: 123456" → 保持 QR_CODE
```

## 🔧 功能特性

### 智能提示标识
- 自动检测的码类型会显示 **"智能"** 绿色标签
- 用户手动选择的类型不显示标签
- 只有 CODE_128 和 QR_CODE 的自动检测会显示标签

### 下拉菜单优化
1. **QR_CODE** - 二维码（最常用）
2. **CODE_128** - Code 128 条形码（次常用）
3. **CODE_39** - Code 39 条形码
4. **EAN_13** - EAN-13 条形码
5. **EAN_8** - EAN-8 条形码
6. **UPC_A** - UPC-A 条形码
7. **ITF_14** - ITF-14 条形码
8. **DATA_MATRIX** - Data Matrix 码
9. **PDF_417** - PDF 417 码

### 动态输入提示
根据当前选择的码类型，输入框会显示相应的提示信息：
- 二维码：通用内容提示
- 条形码：格式要求提示
- 数字条码：位数要求提示

## 📱 使用场景

### 1. 产品编号场景
```
复制 → "PRD2024001" 
自动检测 → CODE_128
```

### 2. 商品条码场景
```
复制 → "6901234567890"
自动检测 → EAN_13
```

### 3. 网址链接场景
```
复制 → "https://mall.example.com/item/12345"
自动检测 → QR_CODE
```

### 4. 库存编号场景
```
复制 → "INVT789XYZ"
自动检测 → CODE_128
```

## 🔄 工作流程

1. **读取剪贴板** → 获取内容
2. **智能分析** → 检测内容特征
3. **类型匹配** → 选择最佳码类型
4. **自动切换** → 更新界面显示
5. **生成预览** → 实时显示码图

## 💡 用户体验优化

- **无感知切换**: 自动检测，用户无需手动选择
- **智能标签**: 明确标识自动检测状态
- **手动覆盖**: 用户可随时手动选择其他类型
- **即时反馈**: 实时更新预览和提示

这个功能大大提升了使用效率，特别是在处理不同类型的编码数据时。
