# 条形码功能测试说明

## 已添加的功能

### 1. 码类型支持
- ✅ QR_CODE - 二维码
- ✅ CODE_128 - Code 128 条形码
- ✅ CODE_39 - Code 39 条形码
- ✅ EAN_13 - EAN-13 条形码 (13位数字)
- ✅ EAN_8 - EAN-8 条形码 (8位数字)
- ✅ UPC_A - UPC-A 条形码 (12位数字)
- ✅ ITF_14 - ITF-14 条形码 (14位数字)
- ✅ DATA_MATRIX - Data Matrix 码
- ✅ PDF_417 - PDF 417 码

### 2. 功能特性
- ✅ 码类型下拉选择器
- ✅ 数据验证 (数字类型条码)
- ✅ 错误提示和引导
- ✅ 实时预览更新
- ✅ 支持图标区分

### 3. 用户体验
- 二维码支持所有文本内容
- 条形码自动数据验证
- 错误状态友好提示
- 内容长度显示
- 剪贴板集成

## 使用说明

1. **选择码类型**: 在输入区域下方选择要生成的码类型
2. **输入内容**: 根据选择的码类型输入相应格式的内容
3. **实时预览**: 内容会实时转换为对应的码
4. **数据验证**: 数字类条码会自动验证输入格式

## 依赖更新

新增依赖:
- `barcode_widget: ^2.2.0` - 条形码生成

## 技术实现

- `CodeGenerator` 统一码生成器
- `CodeType` 枚举定义所有支持的码类型
- `isValidForBarcode()` 静态方法进行数据验证
- 响应式UI适配不同码类型的显示需求
